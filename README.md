# Decide DLT

**Децентрализованная система электронного голосования с динамическим выбором валидаторов**

---

## О проекте

Decide DLT — это демонстрационный прототип системы электронного голосования, разработанный в рамках магистерской диссертации. Система позволяет проводить прозрачные и проверяемые голосования с использованием технологии распределенного реестра (DLT).

**Ключевая особенность:** каждый участник самостоятельно выбирает валидаторов, которые будут подтверждать его голос (персонализированное доверие).

---

## Функциональные возможности

- Регистрация и авторизация пользователей с генерацией ключевой пары Ed25519
- Создание голосований (заголовок, вопрос, варианты ответов, дата/время)
- Голосование с динамическим выбором валидаторов
- Локальная валидация транзакций выбранными валидаторами
- Автоматическое формирование блоков при накоплении 3 транзакций
- P2P-синхронизация в локальной Wi-Fi сети
- Личный кабинет с историей голосов
- Блокчейн-эксплорер для просмотра всех блоков и транзакций
